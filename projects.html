Parallelized SEIR Modelling

For my Parallel Computing course, I developed a parallelized simulation of the SEIR model for the spread of disease through a population. This work presents a general solution to how parallel programming techniques can be applied to different approaches to disease modeling using celluar automata. My work on this was based on an existing model, but instead using a distributed model with MPI combined with a shared memory model using pthreads.
